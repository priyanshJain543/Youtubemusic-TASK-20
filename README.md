# 🎵 YouTube Music Inspired - Web Player

A high-performance, fully responsive music player built with **Vanilla JavaScript, HTML5, and CSS3**. This project focuses on a pixel-perfect UI/UX, featuring a dynamic song list, interactive lyrics card, and a real-time audio visualizer.

## ✨ Key Features
* **Custom Branding**: YouTube Music inspired favicon and tab title implementation.
* **Interactive Lyrics Card**: A unique "Flip Card" animation that reveals 100% full formatted lyrics for every song using `white-space: pre-wrap`.
* **Real-time Visualizer**: Dynamic frequency bars that respond to audio playback.
* **Smart Audio Logic**: Custom-built controls for Play/Pause, Next/Previous, and an interactive Seekbar.
* **Error Handling**: Implemented logic to prevent `404 Not Found` errors by standardizing file naming (e.g., `woh_din.mp3`).

## 📱 Mobile Responsive Solutions (Debugging)
Major layout challenges were solved to ensure a native app-like experience:
* **Scroll Management**: Fixed double-scroll issues by utilizing `overflow-y: auto` on the body and handling fixed player heights.
* **Layout Stacking**: Optimized UI for mobile by transitioning from a side-by-side desktop layout to a vertical stack.
* **Responsive Spacing**: Used `calc()` and specific padding constants to ensure player controls never overlap with the song list on smaller screens.

## 🛠️ Tech Stack
* **Language**: JavaScript (ES6+ DOM Manipulation)
* **Styling**: Pure CSS3 (Flexbox, Grid, and Keyframe Animations)
* **Structure**: Semantic HTML5
* **Hardware**: Developed on ASUS TUF Gaming A15
